# QuickDelivery
Delivery Boy app for **QuickFoodRide**.

## Features:
* **Order Notifications**
* **Order details**
* **Restaurant location on map**
* **Customer location on map**
* **Customer contact details**
* **Restaurant details**
* **Order delivery history**
* **Push Notification**
* **Order status checking**
